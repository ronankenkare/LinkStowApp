Boundary detail

What crosses the system boundary

The context view separates saved user content from temporary network responses and OS-mediated operations.

IN

User input

URL text, link edits, group selections, symbol choices, visibility choices, filters, sorting, and swipe actions enter through SwiftUI.

WEB

Website enrichment

URLSession retrieves a response and HTML. SwiftSoup extracts title, description, icon links, and Open Graph image references.

OS

Device-owned actions

Authentication UI, browser launch, sharing destinations, clipboard access, and QR image handling stay behind Apple platform interfaces.

Data-locality statement: the current source contains no LinkStow account service, cloud database, analytics endpoint, or application backend. Public website content is fetched only to validate or enrich a link.
